# Tips inför checkpoint

Öva på:

    Console.ReadLine
    Console.WriteLine

    Kunna skapa en lista från en sträng mha "Split"

    Parse'a strängar och tecken (int.Parse, char.Parse) 

    Loopa mha "for"

    Lösa problem med en "for"-loop i en annan "for"-loop. T.ex:

        for (int a = 1; a <= 2; a++)
        {
            for (int b = 1; b <= 3; b++)
            {
                Console.WriteLine("[" + a + " " + b + "]");
            }
        }
